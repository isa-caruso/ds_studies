## Featured Project: Autonomous Financial Auditor

This is a high-maturity implementation of a **Hybrid Agentic Architecture** designed to solve real-world financial reconciliation problems.

**Key Highlights:**
* **Hybrid Reasoning:** Uses a local **Llama 3.2 (1B)** for executive synthesis while delegating math and data processing to deterministic Python modules.
* **Engineering Rigor:** Implements **Inversion of Control** to prevent LLM hallucinations, ensuring 100% financial accuracy.
* **Complex UI Handling:** Solves Jupyter/MathJax rendering issues by injecting pre-formatted HTML tables directly into AI-generated reports.

> This project demonstrates the transition from simple prompting to building reliable, scalable AI systems that overcome the limitations of small-parameter local models.
